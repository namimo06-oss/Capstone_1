# Capstone_1
This project analyzes sales data for a fictional business, EmporiUm, using SQL and Excel to support data-driven decision-making. The analysis focuses on the Connecticut territory within the East region, where SQL queries were used in MySQL Workbench to evaluate total revenue, identify the date range of transactions, analyze monthly revenue trends, compare territory performance to the overall region, examine transaction volume and average transaction size by product category, and rank store performance based on total sales. The results of this analysis were further visualized using an Excel PivotChart created from exported query data. This repository includes the SQL script used to generate the analysis, the Excel visualization file, and supporting project materials. A video presentation demonstrating the analysis and results is also included as part of the project requirements.
